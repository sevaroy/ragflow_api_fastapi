🎯 核心特色功能

RAGFlow 深度集成

直接串接 RAGFlow Python API，零中介層架構設計。支持多數據集動態切換，使用 streamlit-chat 專業聊天界面，提供完整的知識庫對話體驗。

RAGAS 全面評估

基於學術級 RAGAS 框架，提供 6 種核心評估指標：忠實度、答案相關性、上下文精確度、召回率、答案相似度和正確性，全面評估 RAG 系統性能。

實時數據視覺化

提供 4 種專業 Plotly 圖表：雷達圖展示綜合表現、趨勢圖追蹤改進軌跡、對比圖分析 A/B 測試、分布圖統計評分分布。

企業級可靠性

完整的錯誤處理機制，包括 EvaluationResult 序列化、NaN 值處理、JSON 數據修復。支持數據持久化存儲和批量評估處理。

自動化評估流程

一鍵啟動完整評估流程，支持聊天記錄自動轉換、測試案例生成、批量評估執行，將評估時間從數小時縮短到 30 秒以內。

智能數據管理

JSON 文件持久化存儲，支持評估歷史追蹤、數據導出管理、損壞文件自動修復，提供完整的評估數據生命周期管理。

# RAGFlow API 直接串接
class RAGFlowOfficialClient:
    def chat_completion(self, chat_id, session_id, question):
        """直接調用 RAGFlow API 進行對話"""
        response = requests.post(
            f"{self.base_url}/v1/chats/{chat_id}/completions",
            headers={"Authorization": f"Bearer {self.api_key}"},
            json={
                "question": question,
                "session_id": session_id,
                "quote": True,
                "stream": False
            }
        )
        return response.json()

from ragas.metrics import (
    faithfulness,        # 忠實度 - 防止 AI 幻覺
    answer_relevancy,    # 答案相關性 - 確保回答精準
    context_precision,   # 上下文精確度 - 優化檢索質量
    context_recall,      # 上下文召回率 - 防止信息遺漏
    answer_similarity,   # 答案相似度 - 與標準答案對比
    answer_correctness   # 答案正確性 - 事實準確性驗證
)

def evaluate_with_ragas(self, test_cases, selected_metrics):
    """使用 RAGAS 進行多維度評估"""
    # 準備評估數據
    data = {
        'question': [case['question'] for case in test_cases],
        'answer': [case['answer'] for case in test_cases],
        'contexts': [case['contexts'] for case in test_cases],
        'ground_truth': [case.get('expected_answer', '') for case in test_cases]
    }
    dataset = Dataset.from_dict(data)
    
    # 選擇並執行評估指標
    metrics = [self.available_metrics[name] for name in selected_metrics]
    evaluation_result = evaluate(dataset, metrics=metrics)
    
    # 處理 EvaluationResult 對象並序列化
    return self.process_evaluation_result(evaluation_result)
